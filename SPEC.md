# PixelViewer - Python 版本规格文档

## 1. 项目概述

**项目名称**: PixelViewer
**项目类型**: Windows 桌面应用 (Python + PyQt6)
**核心功能**: 解码并显示多种格式的 YUV/RGB 原始图像文件，支持自定义分辨率和像素格式
**目标用户**: 视频开发者、图像处理工程师、调试人员

## 3. 功能规格

### 支持的格式

#### RGB 系列
| 格式 | 描述 | 每像素字节 |
|------|------|-----------|
| RGB888 | 24位 RGB | 3 |
| BGR888 | 24位 BGR | 3 |
| RGB565 | 16位 RGB (5-6-5) | 2 |
| BGR565 | 16位 BGR (5-6-5) | 2 |
| XRGB8888 | 32位 ARGB (忽略Alpha) | 4 |
| XBGR8888 | 32位 ABGR (忽略Alpha) | 4 |

#### YUV 系列
| 格式 | 描述 | 采样 |
|------|------|------|
| NV12 | YUV 4:2:0, UV 交错 | 2 平面 |
| NV21 | YUV 4:2:0, VU 交错 | 2 平面 |
| NV16 | YUV 4:2:2, UV 交错 | 2 平面 |
| NV61 | YUV 4:2:2, VU 交错 | 2 平面 |
| NV24 | YUV 4:4:4, UV 交错 | 2 平面 |
| NV42 | YUV 4:4:4, VU 交错 | 2 平面 |

### 核心功能

1. **文件打开**: 打开原始二进制文件
2. **分辨率设置**: 宽度 x 高度输入
3. **格式选择**: 下拉选择像素格式
4. **实时解码**: 解析字节并渲染
5. **缩放控制**: 1x, 2x, 4x, 8x, 适应窗口
6. **拖拽平移**: 鼠标拖拽移动视图
7. **鼠标信息**: 显示坐标和颜色值
